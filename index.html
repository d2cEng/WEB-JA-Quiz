만들고 싶은 단어장 주제를 적고 프롬프트 복사를 누르세요. 복사된 내용을 ChatGPT·Claude·Gemini 등 AI에 붙여넣으면 아래 형식에 맞는 JSON을 만들어 줍니다. 그 결과를 .json 파일로 저장해 📥 가져오기 하면 끝!

  1. 주제를 적고 프롬프트 복사를 누릅니다.
  2. AI(ChatGPT·Claude·Gemini 등)에 붙여넣고 실행합니다.
  3. AI가 준 JSON을 메모장에 붙여넣어 주제.json으로 저장합니다.
  4. 위 📥 가져오기로 그 파일을 불러옵니다.

직접 만들려면 엑셀·메모장 등으로 아래 형식의 .json 파일을 만들어 📥 가져오기로 불러오면 됩니다. 📤 내보내기로 받은 파일을 열어 words 안의 내용만 바꾸는 것도 쉬운 방법입니다.

name단어장 이름 (선택)
kanji간지 표기 (한자 없으면 히라가나와 동일하게)
hiragana읽는 법 (히라가나)
korean한글 뜻 — 여러 뜻은 쉼표(,)로 구분
level레벨 (예: N5, N4) — 자유
pos품사 (1그룹동사·2그룹동사·3그룹동사·い형용사·な형용사·명사·부사·접속사…)
example예문 — 뜻이 여러 개면 쉼표(,)로 뜻과 같은 순서로
{ "version": 1, "name": "JLPT N5 단어장", "words": [ { "id": 1, "kanji": "教える", "hiragana": "おしえる", "korean": "가르치다,알려주다", "level": "N5", "pos": "2그룹동사", "example": "日本語を教える。,道を教える。" } ] }

• id는 생략해도 자동 부여됩니다. • korean과 example의 쉼표 개수를 맞추면 뜻별 예문이 1:1로 연결됩니다. • 예문 구분자가 、·，여도 인식합니다.
