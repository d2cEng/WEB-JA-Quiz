# 일본어 단어 암기 · 시험 (JA Quiz)

**👉 바로 사용하기: [https://d2ceng.github.io/WEB-JA-Quiz/](https://d2ceng.github.io/WEB-JA-Quiz/)**

설치 없이 브라우저에서 바로 쓰는 일본어 단어 학습 웹앱입니다. 위 주소를 열기만 하면 되고,
**`index.html` 파일 하나로도 작동**하므로 카톡 등으로 파일을 보내 오프라인으로 쓸 수도 있습니다.

## 기능
- **다국어(한국어/English)** — 우측 상단에서 앱 언어를 전환합니다. 화면·문법·도움말까지 모두 번역됩니다.
- **도움말(❔)** — 상단의 도움말 버튼으로 각 기능 사용법을 힌트 카드로 한눈에 봅니다.
- **단어 DB** — 단어 목록 검색·정렬·필터(품사/레벨), 체크로 암기 대상 선정, JSON 가져오기/내보내기
- **일정** — 시작일~마지막일에 단어 자동 분배. 평일/주말 하루 분량 따로 설정. 선정 방식: 미리 선정 / 무작위 / 품사별. 학습 현황에 암기·퀴즈로 학습한 **일정 외 학습**도 함께 집계
- **암기(플래시카드)** — ① 뜻+히라가나 → ② 히라가나+간지 → ③ **간지+뜻** → ④ 예문 4단계. 단계 이동 버튼에 실제 단계 이름 표시, 카드에서 **이 단어 빼기**로 아는 단어 제외, 뜻이 여러 개면 뜻별 예문 선택
- **퀴즈** — 4지선다 → 주관식 순서. 문제·정답 방향을 고정하거나 **무작위로 섞어** 출제
- **시험지 인쇄** — 히라가나·간지·한글 중 무작위로 빈칸 처리한 주관식 시험지를 실제 프린터로 출력(정답지 포함 옵션)

학습 진도와 설정은 **브라우저에 자동 저장**되며, 내보내기/가져오기로 다른 기기에 옮길 수 있습니다.

## 여는 법

### 웹에서 바로 열기 (권장)
[https://d2ceng.github.io/WEB-JA-Quiz/](https://d2ceng.github.io/WEB-JA-Quiz/) 주소를 PC·휴대폰 브라우저에서 열면 바로 사용할 수 있습니다.
링크만 공유하면 받는 사람도 똑같이 열어 쓸 수 있고, 자주 쓴다면 브라우저의 **홈 화면에 추가**로 앱처럼 둘 수 있습니다.

### 파일로 공유하고 여는 법 (오프라인)
인터넷 없이 쓰고 싶다면 파일(`index.html`)을 카톡으로 전송한 뒤:
- **PC** — 받은 파일을 더블클릭하면 기본 브라우저에서 열립니다.
- **안드로이드** — 파일을 누른 뒤 "브라우저(Chrome 등)로 열기"를 선택합니다.
- **아이폰** — 파일을 "파일 앱에 저장" 후, 파일 앱에서 눌러 Safari로 엽니다.

> 진도는 기기마다 따로 저장됩니다. 기기를 바꿀 때는 단어 DB 화면의 **내보내기**로
> 백업 파일을 만들고, 새 기기에서 **가져오기** 하면 단어와 진도가 그대로 복원됩니다.

## 내 단어로 교체하기

### 저장소에 준비된 단어장 불러오기
웹(GitHub Pages)으로 열면 **단어 DB** 화면 상단의 **📦 불러오기**에서 이 저장소에 포함된 단어장을
드롭다운으로 골라 바로 사용할 수 있습니다. (불러오면 현재 단어·진도가 교체되므로, 필요하면 먼저 **내보내기**로 백업하세요.)

> 새 단어장을 목록에 추가하려면: ① `data/` 폴더에 `<이름>.json`을 올리고, ② `data/wordbooks.json`의
> `wordbooks` 배열에 `{ "name": "표시 이름", "file": "<이름>.json" }` 항목을 추가하면 됩니다.
> (단일 파일로 오프라인 사용할 때는 이 기능이 숨겨지고, 아래 **가져오기**로 파일을 직접 불러오면 됩니다.)

### 가장 쉬운 방법 — AI로 만들기
**단어 DB → 📄 JSON 단어장 파일 만드는 법**을 펼치면 **🤖 AI로 만들기** 박스가 있습니다.

1. **단어장 주제**를 직접 정해 입력합니다. (예: `JLPT N4 동사`, `비즈니스 일본어`, `여행 회화 단어`)
2. **📋 프롬프트 복사**를 누릅니다.
3. 복사된 프롬프트를 ChatGPT·Claude·Gemini 등 AI에 붙여넣고 실행합니다.
4. AI가 준 JSON을 메모장에 붙여넣어 `주제.json`으로 저장합니다.
5. **📥 가져오기**로 그 파일을 불러오면 내 단어장이 됩니다.

> 프롬프트에는 아래 JSON 형식·필드 규칙이 자동으로 포함되므로, 주제만 정하면 형식에 맞는 단어장이 만들어집니다.

### 직접 만들기
`data/words.sample.json`과 같은 형식의 JSON을 만들어 **단어 DB → 가져오기**로 넣으면 됩니다.

```json
{
  "version": 1,
  "words": [
    {
      "id": 1,
      "kanji": "明ける",
      "hiragana": "あける",
      "korean": "날이 밝다,(기간이) 끝나다",
      "level": "N4",
      "pos": "1그룹동사",
      "example": "夜が明ける。,梅雨が明ける。"
    }
  ]
}
```

- `korean`, `example`은 **쉼표(,)** 로 같은 간지의 여러 뜻과 그에 맞는 예문을 1:1로 적습니다.
- `pos`(품사)는 **한국어**(`1그룹동사`·`2그룹동사`·`3그룹동사`·`い형용사`·`な형용사`·`명사`·`부사`·`접속사`) 또는 **영어**(`Group 1/2/3 verb`·`i-adjective`·`na-adjective`·`noun`·`adverb`·`conjunction`)로 적을 수 있습니다. 대소문자·공백·하이픈은 자유롭게 인식됩니다.
- 엑셀/CSV로 관리 중이라면 위 컬럼(간지·히라가나·한글·레벨·품사·예문)에 맞춰 JSON으로 변환하세요.

## 파일 구성
- `index.html` — 앱 전체(시드 단어 데이터 포함). 웹으로 배포되며, 파일 하나만 보내도 그대로 작동합니다.
- `data/words.sample.json` — 샘플/참고용 단어 데이터 원본
- `.claude/launch.json` — 로컬 미리보기용 정적 서버 설정(배포에 불필요)

## 참고
- 단어 수가 매우 많아지면 단일 파일이 무거워질 수 있습니다. 그때는 단어 데이터를 외부 JSON으로
  분리하는 방식을 고려하세요.
